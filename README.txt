Instrucciones para obtener repositorio desde GitHub donde se encuentran los artefactos (código, formas, evidencias, etc.) :
1. En el directorio en que desea descargar el proyecto ejecute el comando : git clone https://github.com/johannolayaherrera/Ecos_Tarea6.git
2. Ingrese a la ruta \src\Integracion2 y ejecute el comando : mvn clean install

Para revisar el Javadoc 
1. Ejecute el comando : mvn site
1. Ingrese a la carpeta \Integracion2\target\site y seleccione index, luego en el menú Project Documentation seleccione Project Reports y luego la opción JavaDoc
